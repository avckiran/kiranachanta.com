Notable Projects

Project - 1 : ATX Voice

Objective:

ATX voice is created for the people in the city of Austin, TX. It is a platform to share their opinions through a detailed blog. These blog posts should be related to Austin. People can share anything interesting they found in the city. Please checkout here

Tech Stack:

  • Design & Prototype : Adobe Photoshop & XD
  • Backend : NodeJS, Express
  • Database: MongoDB (Atlas cloud )
  • Frontend: React, Redux (state management), Bootstrap 4
  • Deployment: AWS EC2 with nginX on Ubuntu

Features:

  • 100% Original design. Wireframes are created using photoshop and XD
  • Backend is a RESTful API using MongoDB Atlas to store the data
  • JWT has been implemented for Authentication
  • It connects Eventbrite API, Twitter API, DarkSky (weather API), Google maps (location API) to collect different data.
  • Rich Text Editor (or WYSIWYG) QuillJS has been integrated for blog post creation
  • All React components are functional components.
  • Component level state is maintained by React Hooks (useState, useEffect)
  • App level state is maintained by Redux
  • Form validations are handled both at client level and server level
  • Twitter is configured to display only the latest and popular tweets at Austin TX.
  • Events carousel shows next 5 events in Austin
  • Bootstrap4 and FontAwesome 5 are used for styling
  • The Blog post has CRUD operations with authorizations
  • Edit and Delete buttons enabled only for authorized users
  • Only logged in users can like or comment
  • Only the owners of the comments can delete their comments

Notes:

  • Redux Dev tools configuration is open, you can check various actions and state there
  • Please access the complete project files here
  • If you found any bugs, kindly log an issue in the Git repository
